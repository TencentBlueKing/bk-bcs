### 描述
该接口提供版本：v1.0.0+
 

下载配置项的元数据内容信息。

### 输入参数
| 参数名称     | 参数类型     | 必选   | 描述             |
| ------------ | ------------ | ------ | ---------------- |
| biz_id         | string       | 是     | 业务ID     |
| app_id         | string       | 是     | 应用ID     |

### HEADER设置
| 参数名称     | 参数类型     | 必选   | 描述             |
| ------------ | ------------ | ------ | ---------------- |
| X-Bkapi-File-Content-Id         | string       | 是     | 上传文件内容的SHA256值     |

### 调用示例
```json
```

### 响应示例
```json
文件内容
```

### 响应参数说明
上传的配置项的元数据的内容信息。

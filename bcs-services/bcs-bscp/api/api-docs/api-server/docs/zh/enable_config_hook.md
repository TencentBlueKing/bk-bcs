### 描述

该接口提供版本：v1.0.0+

获取服务配置脚本

### 输入参数

| 参数名称 | 参数类型 | 必选 | 描述     |
| -------- | -------- | ---- | -------- |
| biz_id   | uint32   | 是   | 业务ID   |
| app_id   | uint32   | 是   | 应用ID   |
| enable   | Bool     | 是   | 是否启用 |

### 调用示例

```json
{
  "enable": true
}
```



### 响应示例

```json
{
  "data":{} 
}
```

### 响应参数说明

data

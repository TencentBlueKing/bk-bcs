### 描述

该接口提供版本：v1.0.0+

删除模版套餐

### 输入参数

| 参数名称          | 参数类型 | 必选 | 描述       |
| ----------------- | -------- | ---- | ---------- |
| biz_id            | uint32   | 是   | 业务ID     |
| template_space_id | uint32   | 是   | 模版空间ID |
| template_set_id   | uint32   | 是   | 模版套餐ID |
| force             | bool     | 否   | 是否强制删除，如果为true，存在未命名版本服务引用的情况下，会级联删除它们，否则报错 |

### 描述
该接口提供版本：v1.0.0+
 

通过文件导入kv，支持类型[xlsx,xml,json,yaml]

### 输入参数
| 参数名称     | 参数类型     | 必选   | 描述             |
| ------------ | ------------ | ------ | ---------------- |
| biz_id         | string       | 是     | 业务ID     |
| app_id         | string       | 是     | 应用ID     |
|         | byte       | 是     | 文件流    |


### 调用示例
```json
文件内容的文件流
```

### 响应示例
```json
{
  "code": 0,
  "message": ""
}
```

### 描述

该接口提供版本：v1.0.0+

hook版本上线，每个hook只有一个上线的版本，会把之前上线的版本状态设为已下线

### 输入参数

| 参数名称   | 参数类型 | 必选 | 描述   |
| ---------- | -------- | ---- | ------ |
| biz_id     | uint32   | 是   | 业务ID |
| hook_id    | uint32   | 是   | hookID |
| release_id | uint32   | 是   | 版本ID |

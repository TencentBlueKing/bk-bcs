### 描述

该接口提供版本：v1.0.0+

创建服务配置脚本，一个服务只能创建一条记录

### 输入参数

| 参数名称     | 参数类型 | 必选 | 描述                               |
| ------------ | -------- | ---- | ---------------------------------- |
| biz_id       | uint32   | 是   | 业务ID                             |
| app_id       | uint32   | 是   | 应用ID                             |
| pre_hook_id  | uint32   | 否   | 前置脚本id，和后置脚本不能同时为空 |
| post_hook_id | uint32   | 否   | 后置脚本id，和前置脚本不能同时为空 |

### 调用示例

```json
{
    "pre_hook_id": 79,
    "post_hook_id": 79
}
```

### 响应示例

```json
{
    "data": {
      "id" :1 
    }
}
```

### 响应参数说明

| 参数名称 | 参数类型 | 描述 |
| -------- | -------- | ---- |
| id       | uint32   | id   |

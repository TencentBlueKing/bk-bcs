### 输入参数
| 参数名称     | 参数类型     | 必选   | 描述             |
| ------------ | ------------ | ------ | ---------------- |
| biz_id         | uint32       | 是     | 业务ID     |
| app_id         | uint32       | 是     | 应用ID     |
| release_id     | uint32       | 是     | 版本ID     |
| all            | bool         | 是     | 是否全量发布     |
| groups         | array       | 是     | 分组ID列表     |

### 描述

该接口提供版本：v1.0.0+

删除服务模版绑定

### 输入参数

| 参数名称         | 参数类型 | 必选 | 描述                                             |
| ---------------- | -------- | ---- | ------------------------------------------------ |
| biz_id           | uint32   | 是   | 业务ID                                           |
| app_id           | uint32   | 是   | 应用ID                                           |
| binding_id       | uint32   | 是   | 服务模版绑定ID                                   |
| template_set_ids | uint32   | 是   | 模板套餐ID列表，逗号分隔，最多200个，比如"1,2,3" |
